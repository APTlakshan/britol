**Action Plan:**

- Includes: Local keywords (5 suburbs) + USPs (eco-friendly, professional, 12 years) + CTA (Free quotes)

### PRIORITY 2: Schema Markup Updates

**Update LocalBusiness Schema to Include All SE Suburbs:**
```json
"areaServed": [
  {"@type": "City", "name": "Dandenong", "addressRegion": "VIC"},
  {"@type": "City", "name": "Pakenham", "addressRegion": "VIC"},
  {"@type": "City", "name": "Narre Warren", "addressRegion": "VIC"},
  {"@type": "City", "name": "Cranbourne", "addressRegion": "VIC"},
  {"@type": "City", "name": "Berwick", "addressRegion": "VIC"},
  {"@type": "City", "name": "Frankston", "addressRegion": "VIC"},
  // ... add 5-10 more major SE suburbs
]
```

**Add Service Schema:**
```json
{
  "@type": "Service",
  "name": "Office Cleaning",
  "description": "Professional office cleaning in SE Melbourne",
  "areaServed": "South East Melbourne",
  "provider": {"@type": "LocalBusiness", "name": "Britol Group"}
}
```
