### 4. **Schema Markup (Structured Data)**

#### **Local Business Schema**
```json
{
  "@type": "LocalBusiness",
  "areaServed": ["Melbourne", "Sydney", "Brisbane"],
  "serviceArea": "Southeast Australia"
}
```
Benefits:
- Helps Google understand your service areas
- Enables local map results
- Improves knowledge panel visibility
- Shows up in "near me" searches

#### **Organization Schema**
```json
{
  "foundingDate": "2012",
  "numberOfEmployees": "50+",
  "aggregateRating": "4.9/5"
}
```
Benefits:
- Establishes business authority
- Shows rating/reviews in search results
- Displays in Google Knowledge Panel
- Increases click-through rate

#### **Breadcrumb Schema**
```json
{
  "Home" → "Services" → "Gallery" → "Contact"
}
```
Benefits:
- Breadcrumb navigation in search results
- Better UX in SERPs
- Helps Google crawl site structure
